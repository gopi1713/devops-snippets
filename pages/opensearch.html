Check Cluster Health

curl -XGET "http://localhost:9200/_cluster/health?pretty"

Shows the health status of the OpenSearch cluster

List All Indices

curl -XGET "http://localhost:9200/_cat/indices?v"

Shows a list of all indices in the cluster

Delete Index

curl -XDELETE "http://localhost:9200/index_name"

Deletes an index from the cluster

Create Index

curl -XPUT "http://localhost:9200/index_name"

Creates a new index in the cluster

Cluster Stats

curl -XGET "http://localhost:9200/_cluster/stats?human&pretty"

Shows statistics about the cluster

Node Information

curl -XGET "http://localhost:9200/_nodes?pretty"

Shows information about nodes in the cluster

Simple Search

curl -XGET "http://localhost:9200/index_name/_search?q=field:value&pretty"

Performs a simple search query against an index

JSON Search Query

curl -XGET "http://localhost:9200/index_name/_search" -H 'Content-Type: application/json' -d'
{
  "query": {
    "match": {
      "field_name": "value"
    }
  }
}'

Performs a search query using JSON format

Create Index with Mappings

curl -XPUT "http://localhost:9200/index_name" -H 'Content-Type: application/json' -d'
{
  "settings": {
    "number_of_shards": 3,
    "number_of_replicas": 2
  },
  "mappings": {
    "properties": {
      "field1": { "type": "text" },
      "field2": { "type": "keyword" },
      "field3": { "type": "date" }
    }
  }
}'

Creates an index with custom settings and mappings

Index Statistics

curl -XGET "http://localhost:9200/index_name/_stats?pretty"

Shows detailed statistics for a specific index

Shard Allocation

curl -XGET "http://localhost:9200/_cat/shards?v"

Shows the allocation of shards across nodes

Node Tasks

curl -XGET "http://localhost:9200/_tasks?pretty"

Shows currently executing tasks on nodes

Force Merge Index

curl -XPOST "http://localhost:9200/index_name/_forcemerge?max_num_segments=1"

Forces merging of an index to optimize it

Update Index Settings

curl -XPUT "http://localhost:9200/index_name/_settings" -H 'Content-Type: application/json' -d'
{
  "index": {
    "number_of_replicas": 1,
    "refresh_interval": "30s"
  }
}'

Updates settings for an existing index

Index Disk Usage

curl -XGET "http://localhost:9200/_cat/indices?v&h=index,store.size"

Shows disk usage for each index

Cluster Allocation Explained

curl -XGET "http://localhost:9200/_cluster/allocation/explain?pretty"

Explains shard allocation decisions for better debugging

Get Index Mapping

curl -XGET "http://localhost:9200/index_name/_mapping?pretty"

Shows the mapping for a specific index

Reindex Data

curl -XPOST "http://localhost:9200/_reindex" -H 'Content-Type: application/json' -d'
{
  "source": {
    "index": "source_index"
  },
  "dest": {
    "index": "destination_index"
  }
}'

Copies documents from one index to another

Snapshot Repository

curl -XPUT "http://localhost:9200/_snapshot/backup_repository" -H 'Content-Type: application/json' -d'
{
  "type": "fs",
  "settings": {
    "location": "/path/to/backup/directory"
  }
}'

Registers a snapshot repository for backups

Create Snapshot

curl -XPUT "http://localhost:9200/_snapshot/backup_repository/snapshot_name" -H 'Content-Type: application/json' -d'
{
  "indices": "index1,index2",
  "ignore_unavailable": true
}'

Creates a snapshot of specific indices

Restore Snapshot

curl -XPOST "http://localhost:9200/_snapshot/backup_repository/snapshot_name/_restore" -H 'Content-Type: application/json' -d'
{
  "indices": "index1,index2",
  "ignore_unavailable": true
}'

Restores indices from a snapshot

Check Index Existence

curl -I "http://localhost:9200/index_name"

Checks if an index exists (returns HTTP 200 if it does, 404 if not)

Flush Index

curl -XPOST "http://localhost:9200/index_name/_flush"

Flushes an index, ensuring data is persisted to disk
