Update Package Lists

sudo apt update

Updates the package lists for upgrades and new package installations

Upgrade Installed Packages

sudo apt upgrade

Upgrades all installed packages to their latest versions

Install MySQL Server

sudo apt install mysql-server

Installs MySQL server on Ubuntu

Install MySQL Client and Dumper

sudo apt install mysql-client

Installs MySQL client with mysqldump tool included

Install PHP 8.1 for Magento

sudo apt install software-properties-common
sudo add-apt-repository ppa:ondrej/php
sudo apt update
sudo apt install php8.1 php8.1-cli php8.1-fpm php8.1-common php8.1-mysql php8.1-zip php8.1-gd php8.1-mbstring php8.1-curl php8.1-xml php8.1-bcmath php8.1-intl php8.1-soap php8.1-xsl php8.1-sockets

Installs PHP 8.1 with all extensions required for Magento

Install Elasticsearch 7.x for Magento

wget -qO - https://artifacts.elastic.co/GPG-KEY-elasticsearch | sudo apt-key add -
sudo apt install apt-transport-https
echo "deb https://artifacts.elastic.co/packages/7.x/apt stable main" | sudo tee /etc/apt/sources.list.d/elastic-7.x.list
sudo apt update
sudo apt install elasticsearch

Installs Elasticsearch 7.x required for Magento 2.4.x

Install OpenSearch for Magento

wget -qO - https://artifacts.opensearch.org/publickeys/opensearch.pgp | sudo apt-key add -
echo "deb https://artifacts.opensearch.org/releases/bundle/opensearch/1.x/apt stable main" | sudo tee /etc/apt/sources.list.d/opensearch-1.x.list
sudo apt update
sudo apt install opensearch

Installs OpenSearch for Magento 2.4.4+

Install Composer 2 (for Magento)

curl -sS https://getcomposer.org/installer | sudo php -- --install-dir=/usr/local/bin --filename=composer --version=2.5.4

Installs Composer 2.5.4, easily adjustable to other versions for Magento compatibility

Install Docker

curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh ./get-docker.sh 
sudo usermod -aG docker $USER
newgrp docker

Installs Docker, adds current user to the docker group, and applies group changes

Install Magento Command Line Tool

composer global require magento/magento-cloud-cli

Installs the Magento Cloud CLI tool

Install n98-magerun2 (Magento CLI)

curl -O https://files.magerun.net/n98-magerun2.phar
chmod +x n98-magerun2.phar
sudo mv n98-magerun2.phar /usr/local/bin/n98-magerun2

Installs n98-magerun2, a powerful CLI tool for Magento 2

Install Redis with PHP Extension

sudo apt install redis-server php-redis

Installs Redis server with PHP extension for caching

Install Varnish for Magento

sudo apt install varnish

Installs Varnish Cache for Magento frontend acceleration

Configure PHP Memory Limit for Magento

sudo sed -i 's/memory_limit = .*/memory_limit = 2G/' /etc/php/7.4/fpm/php.ini
sudo sed -i 's/memory_limit = .*/memory_limit = 2G/' /etc/php/7.4/cli/php.ini

Sets PHP memory limit to 2GB for Magento (adjust PHP version as needed)

Install Nginx with Magento Config

sudo apt install nginx
sudo wget -O /etc/nginx/sites-available/magento https://raw.githubusercontent.com/magenx/Magento-nginx-config/master/magento2/sites-available/magento2.conf

Installs Nginx with a ready-to-use Magento configuration
