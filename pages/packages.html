Package Installation

Update Package Lists

sudo apt update

Updates the package lists for upgrades and new package installations

Upgrade Installed Packages

sudo apt upgrade

Upgrades all installed packages to their latest versions

Install MySQL Server

sudo apt install mysql-server

Installs MySQL server on Ubuntu

Install MySQL Client and Dumper

sudo apt install mysql-client

Installs MySQL client with mysqldump tool included

Install Apache Web Server

sudo apt install apache2

Installs Apache HTTP server on Ubuntu

Install Nginx Web Server

sudo apt install nginx

Installs Nginx web server on Ubuntu

Install Redis Server

sudo apt install redis-server

Installs Redis server on Ubuntu

Install PHP and Common Extensions

sudo apt install php php-cli php-fpm php-mysql php-curl php-gd php-mbstring php-xml php-zip

Installs PHP with commonly used extensions

Install Git

sudo apt install git

Installs Git version control system

Install OpenSearch Dependencies

sudo apt install apt-transport-https ca-certificates gnupg2

Installs dependencies required for OpenSearch

Install Composer (PHP Package Manager)

curl -sS https://getcomposer.org/installer | php sudo mv composer.phar /usr/local/bin/composer

Downloads and installs Composer globally

Install MyDumper

sudo apt install mydumper

Installs MyDumper, a high-performance MySQL backup tool
