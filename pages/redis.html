Start Redis Server

redis-server

Starts the Redis server

Stop Redis Server

redis-cli shutdown

Stops the Redis server

Flush All Data

redis-cli flushall

Deletes all keys in all databases

Flush Current Database

redis-cli flushdb

Deletes all keys in the current database

Backup Database

redis-cli save

Creates a snapshot of the database

Restore Database

redis-cli restore

Restores the database from a dump file

Monitor Redis Activity

redis-cli monitor

Streams back every command processed by the server in real-time

Check Server Status

redis-cli ping

Checks if the server is running
