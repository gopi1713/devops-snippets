Start Redis Server

redis-server

Starts the Redis server

Stop Redis Server

redis-cli shutdown

Stops the Redis server

Flush All Data

redis-cli flushall

Deletes all keys in all databases

Flush Current Database

redis-cli flushdb

Deletes all keys in the current database

Backup Database

redis-cli save

Creates a snapshot of the database

Restore Database

redis-cli restore

Restores the database from a dump file

Monitor Redis Activity

redis-cli monitor

Streams back every command processed by the server in real-time

Check Server Status

redis-cli ping

Checks if the server is running

Connect to Remote Redis Server

redis-cli -h host -p port -a password

Connects to a Redis server at specified host, port with authentication

Set Key

SET key value

Sets the string value of a key

Get Key

GET key

Gets the value of a key

Delete Key

DEL key

Deletes a key

Check Key Existence

EXISTS key

Determines if key exists

Set Expiration for Key

EXPIRE key seconds

Sets a key's time to live in seconds

Get Key Time to Live

TTL key

Returns the remaining time to live of a key

Show All Keys

KEYS pattern

Finds all keys matching the specified pattern

Redis Info

INFO

Shows information and statistics about the server

Redis Memory Stats

INFO memory

Shows memory-related information and statistics

Redis Server Config

CONFIG GET *

Gets the value of all configuration parameters

Change Config

CONFIG SET parameter value

Sets a configuration parameter to the given value

Increment Value

INCR key

Increments the integer value of a key by one

Decrement Value

DECR key

Decrements the integer value of a key by one

Hash Set

HSET key field value

Sets field in the hash stored at key to value

Hash Get

HGET key field

Gets the value of a hash field

List Push

LPUSH key value [value ...]

Inserts values at the head of a list

List Range

LRANGE key start stop

Gets elements from a list within the specified range

Set Add

SADD key member [member ...]

Adds members to a set

Set Members

SMEMBERS key

Returns all members of a set

Sorted Set Add

ZADD key score member

Adds a member with score to a sorted set

Sorted Set Range

ZRANGE key start stop [WITHSCORES]

Returns elements in a sorted set within a range

DB Size

DBSIZE

Returns the number of keys in the current database

Slow Log

SLOWLOG GET [count]

Shows the slow log entries

Client List

CLIENT LIST

Returns information and statistics about client connections
