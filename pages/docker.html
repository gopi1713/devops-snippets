List all containers

docker ps -a

Shows all containers, including stopped ones

Build an image

docker build -t image-name:tag .

Builds a Docker image from a Dockerfile in the current directory

Run a container

docker run -d -p 8080:80 --name my-container image-name:tag

Runs a container in detached mode with port mapping

Stop a container

docker stop container-name-or-id

Stops a running container

Remove a container

docker rm container-name-or-id

Removes a stopped container

List all images

docker images

Lists all Docker images

Remove an image

docker rmi image-name:tag

Removes a Docker image

Exec into container

docker exec -it container-name-or-id bash

Executes an interactive bash shell inside a running container

View container logs

docker logs -f container-name-or-id

Shows logs for a container with follow option

Container resource usage

docker stats

Shows live resource usage statistics for running containers

Prune unused resources

docker system prune -a

Removes all unused containers, networks, images and volumes

Show image history

docker history image-name:tag

Shows the history of an image with layers information

Inspect container

docker inspect container-name-or-id

Returns low-level information on Docker objects

Container diff

docker diff container-name-or-id

Shows file system changes in the container

Create volume

docker volume create volume-name

Creates a volume for persistent data storage

List volumes

docker volume ls

Lists all volumes

Create network

docker network create network-name

Creates a user-defined network

List networks

docker network ls

Lists all networks

Connect container to network

docker network connect network-name container-name-or-id

Connects a container to a network

Copy files to container

docker cp /local/path/file.txt container-name-or-id:/container/path/

Copies files/folders from local filesystem to a container

Copy files from container

docker cp container-name-or-id:/container/path/file.txt /local/path/

Copies files/folders from a container to local filesystem

Export container filesystem

docker export container-name-or-id > container.tar

Exports a container's filesystem as a tar archive

Save image

docker save -o image.tar image-name:tag

Saves an image to a tar archive

Load image

docker load -i image.tar

Loads an image from a tar archive

Tag image

docker tag source-image:tag target-image:tag

Creates a tag TARGET_IMAGE that refers to SOURCE_IMAGE

Push image to registry

docker push image-name:tag

Pushes an image to a registry

Pull image from registry

docker pull image-name:tag

Pulls an image from a registry

Login to registry

docker login registry-url

Logs in to a Docker registry

Docker compose up

docker-compose up -d

Creates and starts containers defined in docker-compose.yml

Docker compose down

docker-compose down

Stops and removes containers, networks, volumes defined in docker-compose.yml
