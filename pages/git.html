Initialize Repository

git init

Creates an empty Git repository or reinitialize an existing one

Clone Repository

git clone https://github.com/username/repository.git

Creates a copy of an existing Git repository

Create and Switch to New Branch

git checkout -b branch-name

Creates and switches to a new branch

Add Files

git add .

Adds all changed files to staging area

Commit Changes

git commit -m "Commit message"

Records changes to the repository with a message

Push to Remote

git push origin branch-name

Updates remote refs along with associated objects

Pull from Remote

git pull origin branch-name

Fetches from and integrates with another repository or branch

View Commit History

git log

Shows the commit logs

Rebase Branch

git rebase main

Reapply commits from your branch on top of the main branch

Interactive Rebase

git rebase -i HEAD~3

Interactive rebase for the last 3 commits to squash, edit, or reorder them

Cherry-pick Commit

git cherry-pick commit-hash

Apply changes from a specific commit to the current branch

Stash Changes

git stash

Temporarily save changes that are not ready to be committed

Apply Stash

git stash apply

Apply the most recent stashed changes without removing them from stash

List Stashes

git stash list

List all stashed changesets

Reset Hard

git reset --hard HEAD~1

Reset to the previous commit, discarding all changes

View File Changes

git diff filename

Show changes between working directory and staging area

Amend Commit

git commit --amend

Edit the last commit message or add forgotten files

Force Push (Use with Caution)

git push --force-with-lease origin branch-name

Force push changes but abort if the remote branch has been updated

Create a Tag

git tag -a v1.0.0 -m "Version 1.0.0"

Create an annotated tag for a release version

Push Tags

git push --tags

Push all tags to the remote repository

Show Branches Merged to Main

git branch --merged main

List all branches that have been merged into main branch

Ignore File Mode Changes

git config core.fileMode false

Ignore file permission changes in Git

Set Global Git User

git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

Set your name and email for all Git repositories

View Remote Info

git remote -v

Show remote repositories and their URLs

Checkout Pull Request (GitHub)

git fetch origin pull/123/head:pr-123
git checkout pr-123

Check out a specific pull request locally (123 is the PR number)

Clean Untracked Files

git clean -fd

Remove untracked files and directories (use with caution)
