Cache Flush

php bin/magento cache:flush

Flushes Magento cache storage

Reindex

php bin/magento indexer:reindex

Reindexes all Magento indexes

Clean Cache

php bin/magento cache:clean

Cleans cache type(s)

Deploy Static Content

php bin/magento setup:static-content:deploy -f

Deploys static view files

Upgrade Magento

php bin/magento setup:upgrade

Upgrades Magento application, DB data, and schema

Compile Code

php bin/magento setup:di:compile

Generates DI configuration and all non-existing interceptors and factories

Enable/Disable Module

php bin/magento module:enable VendorName_ModuleName

Enables or disables a module

List Modules

php bin/magento module:status

Displays the status of modules
