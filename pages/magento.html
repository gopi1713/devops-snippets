Cache Flush

php bin/magento cache:flush

Flushes Magento cache storage

Reindex

php bin/magento indexer:reindex

Reindexes all Magento indexes

Clean Cache

php bin/magento cache:clean

Cleans cache type(s)

Deploy Static Content

php bin/magento setup:static-content:deploy -f

Deploys static view files

Upgrade Magento

php bin/magento setup:upgrade

Upgrades Magento application, DB data, and schema

Compile Code

php bin/magento setup:di:compile

Generates DI configuration and all non-existing interceptors and factories

Enable/Disable Module

php bin/magento module:enable VendorName_ModuleName

Enables or disables a module

List Modules

php bin/magento module:status

Displays the status of modules

Enable Maintenance Mode

php bin/magento maintenance:enable

Puts the application into maintenance mode

Disable Maintenance Mode

php bin/magento maintenance:disable

Takes the application out of maintenance mode

Clean Generated Code

rm -rf generated/code/* generated/metadata/*

Manually removes generated code and metadata

Clear View Files Cache

php bin/magento cache:clean layout block_html full_page

Cleans layout, block HTML, and page cache

Run Cron

php bin/magento cron:run

Runs Magento cron jobs

Reset Admin Password

php bin/magento admin:user:create --admin-user=username --admin-password=password --admin-email=email@example.com --admin-firstname=FirstName --admin-lastname=LastName

Creates a new admin user or resets password if user exists

Show Configuration Value

php bin/magento config:show path/to/config

Shows the configuration value for a specific path

Set Configuration Value

php bin/magento config:set path/to/config value

Sets the configuration value for a specific path

List All Configuration Paths

php bin/magento config:show --all

Lists all available configuration paths

Enable Display Errors

php bin/magento deploy:mode:set developer

Sets Magento to developer mode for debugging

Set Production Mode

php bin/magento deploy:mode:set production

Sets Magento to production mode for best performance

Check Current Mode

php bin/magento deploy:mode:show

Displays the current deployment mode

Theme Deployment

php bin/magento setup:static-content:deploy -f --theme=Frontend/vendor/theme

Deploys static content for a specific theme

Clean Media Cache

php bin/magento catalog:images:resize

Resizes catalog images

Run Setup Upgrade

php bin/magento setup:upgrade --keep-generated

Upgrades database schema without removing generated files

Deploy Static Content for Locales

php bin/magento setup:static-content:deploy en_US fr_FR de_DE -f

Deploys static view files for specific locales
