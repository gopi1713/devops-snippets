Docker

List all containers

docker ps -a

Shows all containers, including stopped ones

Build an image

docker build -t image-name:tag .

Builds a Docker image from a Dockerfile in the current directory

Kubernetes

Get all pods

kubectl get pods

Lists all pods in the current namespace

Describe pod

kubectl describe pod [pod-name]

Shows detailed information about a pod

Git

Create and switch to new branch

git checkout -b branch-name

Creates a new branch and switches to it

Push to remote

git push origin branch-name

Pushes local commits to the remote repository

Magento

Cache Flush

php bin/magento cache:flush

Flushes Magento cache storage

Reindex

php bin/magento indexer:reindex

Reindexes all Magento indexes

Deploy Static Content

php bin/magento setup:static-content:deploy -f

Deploys static view files

MySQL

Connect to MySQL

mysql -u username -p database_name

Connect to a MySQL database

Export Database

mysqldump -u username -p database_name > backup.sql

Export a database to an SQL file

Import Database

mysql -u username -p database_name < backup.sql

Import a database from an SQL file

OpenSearch

Check Cluster Health

curl -XGET "http://localhost:9200/_cluster/health?pretty"

Retrieves OpenSearch cluster health status

View Indices

curl -XGET "http://localhost:9200/_cat/indices?v"

Lists all indices in the cluster

Delete Index

curl -XDELETE "http://localhost:9200/index_name"

Deletes an index

Redis

Connect to Redis CLI

redis-cli

Start the Redis command line interface

Flush All Keys

redis-cli FLUSHALL

Delete all keys from all Redis databases

Monitor Redis Commands

redis-cli MONITOR

Stream all commands processed by the Redis server in real-time
